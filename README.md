## SwiftRoute GPS

This is my **SwiftRoute GPS** web application project, built with HTML, CSS, and JavaScript.  
It includes:

- Real-time GPS location tracking
- Interactive Mapbox map
- Marker showing your exact location
- Status messages for location access

## Live Demo

[You-are-here](https://harpreetkaur2201.github.io/GPS/)

## Features

- Track your current location on a map
- Click a button to update location in real-time
- Error handling when location cannot be accessed
